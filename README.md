# AI FAQ Chatbot

An AI-powered FAQ chatbot that answers user questions using semantic search and sentence embeddings.

The chatbot uses a Sentence Transformer model to understand the meaning of user questions and retrieve the most relevant answer from a knowledge base.

---

# Features

- AI-powered semantic search
- Sentence embeddings using Sentence Transformers
- Cosine similarity for matching questions
- Interactive chat interface using Streamlit
- Confidence threshold for better answers
- Category-based knowledge base
- Easy to extend with new FAQs

---

# Technologies Used

- Python
- Streamlit
- Sentence Transformers
- HuggingFace Transformers
- Scikit-learn
- NumPy
- Pandas
